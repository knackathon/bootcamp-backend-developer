# Back-End Development Bootcamp - Knackathon 2021

**Instructor**: [Alejandro Franco](https://alejandrofran.co)

Learn software engineering techniques, technologies and tools for back-end development in Node.js by building a Business Directory REST API that could power a website like Yelp.com or yellowpages.com

![api diagram](./assets/api-diagram.png)
